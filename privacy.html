Política de Privacidad

Esta aplicación utiliza la API oficial de TikTok para autenticar a los usuarios y realizar acciones automatizadas. No almacenamos, vendemos ni compartimos tus datos personales o credenciales de TikTok.

Todos los datos se procesan de forma segura y se utilizan únicamente para las funciones de automatización previstas dentro de n8n. No se utilizan herramientas de análisis ni de seguimiento de terceros.
